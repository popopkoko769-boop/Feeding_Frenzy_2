+# Feeding Frenzy 2 - Arcade Prototype
+
+A 2D arcade-style game inspired by *Feeding Frenzy 2*. Eat smaller fish to grow, avoid bigger fish, and collect power-ups to survive longer.
+
+## How to Run
+
+1. Open `index.html` in a modern web browser.
+2. Click **Start Game** from the main menu.
+
+> Tip: If your browser blocks local file access for any reason, run a quick local server:
+>
+> ```bash
+> python3 -m http.server
+> ```
+>
+> Then open `http://localhost:8000` in your browser.
+
+## Features
+
+- Mouse or keyboard (WASD / Arrow) movement.
+- Growth system and tiered fish sizes.
+- Speed Boost + Shield power-ups.
+- Score, lives, and growth bar UI.
+- Main menu and game over screens.
